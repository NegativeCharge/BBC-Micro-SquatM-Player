# BBC Micro Squeeker Plus 1-bit Beeper Engine Player

- Squeeker Plus beeper music engine
- Ported to the BBC Micro by Negative Charge 06'25

The track currently needs to be included at the bottom of main.6502 (samples are in the tracks directory)

Squeeker Plus tracks can be composed in 1Tracker (https://shiru.untergrund.net/software.shtml) - you will need to export the track in Atari ca65 format, and manually convert to BeebAsm format (see samples for changes required).

SSD file for emulators/hardware: https://github.com/NegativeCharge/BBC-Micro-SquatM-Player/blob/master/SquatM_Beeper_Engine.ssd?raw=true
